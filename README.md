# Personal Link Hub (Shaxsiy Havolalar Markazi)

Bu loyiha Python (Flask), HTML va CSS yordamida yaratilgan zamonaviy, Glassmorphism uslubidagi portfolio saytidir.

## Xususiyatlari
- **Zamonaviy Dizayn**: Dark Purple/Black mavzusi, Neon effektlar.
- **Modal Tizimi**: Instagram kabi havolalar ichida qo'shimcha menyu ochish imkoniyati.
- **Bento Grid**: Chiroyli va tartibli karta ko'rinishi.
- **Moslashuvchan**: Telefon va kompyuter ekranlariga moslashadi.
- **Oson Sozlash**: Barcha ma'lumotlar `links.json` faylidan boshqariladi.

## Fayllar
- `app.py` - Serverni ishga tushirish fayli.
- `links.json` - Ism, rasm va havolalarni o'zgartirish joyi.
- `templates/index.html` - Sayt ko'rinishi kodi.
- `static/style.css` - Dizayn va ranglar.

## Ishga Tushirish

1. Python o'rnatilganligiga ishonch hosil qiling.
2. Terminalda quyidagi buyruqni bering:
   ```bash
   python app.py
   ```
3. Brauzerda [http://127.0.0.1:5000](http://127.0.0.1:5000) manzilini oching.
